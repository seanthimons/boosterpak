# Skimr preset with numeric helpers

Skimr preset with numeric helpers

## Usage

``` r
my_skim(data, ...)
```

## Arguments

- data:

  Data frame to skim.

- ...:

  Additional arguments passed to
  [`skimr::skim()`](https://docs.ropensci.org/skimr/reference/skim.html).

## Value

A skimr summary.
