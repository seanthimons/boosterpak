# Demote a user pack to project scope

Demote a user pack to project scope

## Usage

``` r
demote_pack(name, root = ".", overwrite = FALSE, verbose = NULL)
```

## Arguments

- name:

  Pack name.

- root:

  Project root.

- overwrite:

  Whether to replace an existing project-scope pack.

- verbose:

  Whether to print routine summaries.

## Value

Path to the copied project-scope pack, invisibly.
