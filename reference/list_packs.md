# List available packs

List available packs

## Usage

``` r
list_packs(scope = NULL, root = ".", verbose = NULL)
```

## Arguments

- scope:

  Optional scope filter: `"project"`, `"user"`, or `"builtin"`.

- root:

  Project root.

- verbose:

  Whether to print a routine summary.

## Value

A data frame of available packs, invisibly.
