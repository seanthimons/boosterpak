# Materialize a helper function

Materialize a helper function

## Usage

``` r
add_function(name, root = ".", overwrite = FALSE, verbose = NULL)
```

## Arguments

- name:

  Function catalog name.

- root:

  Project root.

- overwrite:

  Whether to replace an existing materialized file.

- verbose:

  Whether to print a routine summary.

## Value

Materialized file path, invisibly.
