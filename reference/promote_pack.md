# Promote a project pack to user scope

Promote a project pack to user scope

## Usage

``` r
promote_pack(name, root = ".", overwrite = FALSE, verbose = NULL)
```

## Arguments

- name:

  Pack name.

- root:

  Project root.

- overwrite:

  Whether to replace an existing user-scope pack.

- verbose:

  Whether to print routine summaries.

## Value

Path to the copied user-scope pack, invisibly.
